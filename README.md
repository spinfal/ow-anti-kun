# OWアンチくん (OW Counter Picker)

「ウィンストンにリーパー出したいけど、ファラいるからきついかも…何を出せばいいかわからん…！」  
そんなオーバーウォッチプレイヤーのために開発された、敵の編成を入れるだけで「おすすめのヒーロー」を表示するツールです。

追加してほしいアンチデータ、不具合等ありましたら、ボ林(https://x.com/boba_yashi)にDMください。

## 🚀 特徴

* **リアルタイムランキング表示**: 1キャラ選択するたびにおすすめアンチピックをすぐさま表示します。なので1キャラからアンチピックの確認に使えます。
* **ロールキュー (5vs5) 完全対応**: 現在の標準ルールに基づいた編成制限（タンク1/ダメージ2/サポート2）がデフォルトですが、ロール制限を撤廃できます。
* **詳細データベース閲覧**: 各ヒーローに設定しているアンチピック、どの構成（ダイブ・ラッシュ・ポーク）に強いのか、を確認できます。改良の余地ありまくり。編集できるようにするかも。

## 🛠 使い方

1.  **敵チームを選択**: 画面中央のグリッドから、敵ヒーローを最大5人までクリックして選択します。
2.  **推奨アンチを確認**: 右側(スマホだと下かも)の「推奨アンチピック」エリアに、各ロールごと、スコアが高い順にヒーローが表示されます。下矢印を選択することで全ヒーロー確認することができます。
3.  **根拠を表示**: 各カードには「ザリアに強い」「ポークに弱い」といった根拠が表示されるため、状況判断の参考にしてください。
